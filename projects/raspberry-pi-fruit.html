Raspberry Pi Fruit Segregation System

A complete embedded system for real-time fruit classification and physical segregation on a conveyor using a Raspberry Pi and a low-cost camera.

Embedded Linux Lightweight CNNs Mechatronics
View on GitHub

Problem

Many small farms and processing units rely on manual fruit inspection and sorting, which is labor-intensive and inconsistent. Low-cost embedded vision can automate basic grading and segregation tasks, but must run within tight compute and power budgets.

Approach

  • Capture top-down images of fruit items on a small conveyor belt using a Raspberry Pi camera module.
  • Train a lightweight CNN classifier to distinguish fruit type / quality categories.
  • Deploy the model on the Pi using optimized inference.
  • Trigger actuators via GPIO to direct items into bins based on the classifier output.

Results

The system achieves real-time inference on the Raspberry Pi with acceptable latency for conveyor speeds used in the prototype. Technology demonstrator for a field level prototype.

Tech Stack

  • Raspberry Pi (Linux), Python
  • OpenCV for image capture and preprocessing
  • Lightweight CNN (PyTorch)
  • GPIO control for actuators
