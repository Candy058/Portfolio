Areca Nut Segmentation, Classifcation & Surface Characterization

Vision-based pipeline for segmenting individual areca nuts, classifying quality grades, and analyzing surface characteristics.

View on GitHub Read report (PDF)

Problem

Areca nuts exhibit significant variation in size, shape, color, and surface texture depending on maturity and quality grade. Manual inspection is labor-intensive and inconsistent, especially in high-throughput processing environments.

The goal of this project was to design a vision-based pipeline capable of: Segmenting individual areca nuts from conveyor imagery, Classifying nuts into quality categories, Extracting surface-level characteristics (texture, defects, discoloration) and Enabling automated, real-time sorting decisions.

Approach

  • Capture top-view conveyor imagery using a fixed camera setup under controlled lighting.
  • Train an instance segmentation model (Mask-RCNN) to isolate individual nuts.
  • Crop segmented instances and feed them into a classification model for grade prediction.
  • Extract surface descriptors and integrate the pipeline into a near real-time inference loop suitable for embedded deployment.

Results

The system successfully isolates individual areca nuts under moderate occlusion and variable lighting conditions. Instance segmentation enables accurate per-object processing. Surface feature extraction improves separation between visually similar grades. The modular pipeline allows easy retraining for new quality categories. Designed with deployment constraints in mind for edge hardware environments. The project demonstrates how combining learned features with engineered surface descriptors improves robustness in agricultural quality assessment tasks.

Tech Stack

  • Python, PyTorch
  • Active depth camera (D435i)
  • Open 3D, OpenCV
