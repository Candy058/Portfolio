Soft Pneumatic Actuator Simulation

Finite element models of soft pneumatic actuators that capture nonlinear material behavior and pressure-driven deformation, enabling rapid iteration on actuator design.

Finite Element Analysis Hyperelastic Materials Design Iteration
View on GitHub

Problem

Soft pneumatic actuators offer compliant, safe interaction but are challenging to design due to their nonlinear, highly coupled behavior. Building and testing physical prototypes is time- and material-intensive.

The aim of this project is to build simulation models that can predict actuator deformation under pressure, guiding geometry and material choices before fabrication.

Approach

  • Model actuator geometries (e.g., fiber-reinforced segments, bellows) in a CAD tool and import into ABAQUS/ANSYS.
  • Assign hyperelastic material models (e.g., Mooney–Rivlin, Ogden) fit to experimental stress–strain data.
  • Apply internal pressure loads and appropriate boundary conditions to emulate mounting and constraints.
  • Run static and quasi-static analyses to obtain deformation profiles, tip displacement, and stress distributions.

Results

Simulated deformations qualitatively match the behavior of fabricated prototypes, particularly in predicted bending directions and approximate tip deflections. Stress maps provide insight into regions prone to failure, informing reinforcement and wall thickness decisions.

Tech Stack

  • ABAQUS / ANSYS for finite element analysis
  • Python scripting for parametric studies
  • CAD tools (e.g., SolidWorks, Fusion 360)
